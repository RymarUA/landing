# Реферальна програма через Sitniks Open API

## Готово! Ніяких змін не потрібно

Система працює через існуючий Sitniks Open API без жодних модифікацій.

## Як це працює:

### 1. Генерація реферального коду
- Frontend генерує унікальний код: `REF_ABC123_1X2B3C`
- Код зберігається в полі `comment` клієнта: `"Referral code: REF_ABC123_1X2B3C"`

### 2. Пошук реферера
- Frontend завантажує всіх клієнтів через `/open-api/clients?limit=100`
- Шукає реферера по `comment` або `customFields`

### 3. Застосування реферала
- Записує в `comment` запрошеного: `"Referred by: REF_ABC123_1X2B3C"`

## Переваги:
✅ Ніякого SQL
✅ Ніяких змін в Sitniks
✅ Працює через існуючий Open API
✅ Всі дані в CRM для аналітики
