# 🛡️ Адмін панель FamilyHub Market

## 📋 Огляд

Адмін панель дозволяє керувати замовленнями, клієнтами та налаштуваннями магазину. Доступна за адресою `/admin` з паролем.

## 🔐 Доступ

### URL
```
https://familyhub.com.ua/admin
```

### Автентифікація
```env
NEXT_PUBLIC_ADMIN_PASSWORD=your_secure_password
```

### API перевірка
```
GET /api/auth/check-admin
```

## 🎯 Функціонал

### 1. Панель керування
- **Статистика замовлень** - кількість, сума, статуси
- **Активні користувачі** - онлайн, нові реєстрації
- **Популярні товари** - найбільш куповані
- **Система рефералів** - запрошення та бонуси

### 2. Управління замовленнями
- **Перегляд замовлень** - детальна інформація
- **Зміна статусів** - Новий → Обробка → Доставка → Завершено
- **Відстеження ТТН** - додати номер для клієнта
- **Скасування** - повернення товарів

### 3. Управління клієнтами
- **Пошук клієнтів** - по email, телефону, імені
- **Історія замовлень** - всі покупки клієнта
- **Реферальна статистика** - хто кого запросив
- **Блокування** - обмеження доступу

### 4. Управління товарами
- **Додавання товарів** - нові позиції в каталозі
- **Редагування** - ціни, описи, зображення
- **Залишки** - контроль кількості
- **Знижки** - акції та промокоди

## 🔧 Технічна реалізація

### Frontend (Next.js)
```
/app/admin/
├── page.tsx              # Головна сторінка
├── orders/
│   ├── page.tsx         # Список замовлень
│   └── [id]/page.tsx    # Деталі замовлення
├── customers/
│   ├── page.tsx         # Список клієнтів
│   └── [id]/page.tsx    # Деталі клієнта
└── settings/
    └── page.tsx         # Налаштування
```

### API Routes
```
/api/admin/
├── orders/              # CRUD замовлень
├── customers/           # CRUD клієнтів
├── products/            # CRUD товарів
├── stats/               # Статистика
└── auth/                # Автентифікація
```

### База даних
```
orders          - Замовлення
customers       - Клієнти (синхронізація з Sitniks)
products        - Товари
admin_logs      - Логи дій адміна
```
