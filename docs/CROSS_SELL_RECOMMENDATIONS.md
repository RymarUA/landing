# Система кросс-селл рекомендаций "З цим купляють"

## Обзор

Новая умная система рекомендаций предлагает дешевые товары с высокой вероятностью покупки вместо случайных или дорогих товаров.

## Как это работает

### 1. Фильтрация товаров

- **Максимальная цена**: 500 грн (настраивается)
- **Исключение**: товары, которые уже в корзине
- **Наличие**: только товары со складом > 0
- **Сортировка**: рейтинг → количество отзывов → цена (сначала дешевле)

### 2. Алгоритм сортировки

```typescript
.sort((a, b) => {
  // 1. Приоритет по рейтингу
  const ratingDiff = b.rating - a.rating;
  if (Math.abs(ratingDiff) > 0.1) return ratingDiff;
  
  // 2. Приоритет по количеству отзывов
  const reviewsDiff = b.reviews - a.reviews;
  if (reviewsDiff !== 0) return reviewsDiff;
  
  // 3. Чем дешевле, тем лучше
  return a.price - b.price;
})
```

### 3. UI улучшения

- **Скелетоны** во время загрузки
- **Бейджи скидок** для выгодных предложений
- **Рейтинг** в виде звезд
- **Hover эффекты** для карточек
- **Адаптивный заголовок** в зависимости от цены

## Преимущества

1. **Выше конверсия** - дешевые товары покупают легче
2. **Увеличение среднего чека** - импульсивные покупки
3. **Отсутствие дублирования** - не предлагает товары из корзины
4. **Умная сортировка** - популярные и высокооцененные товары первыми
5. **Визуальные триггеры** - бейджи скидок и рейтинги

## Результат

Новая система фокусируется на товарах, которые:
- ✅ Дешевле 500 грн
- ✅ Имеют высокий рейтинг
- ✅ Не дублируют товары в корзине
- ✅ Имеют хорошие отзывы
- ✅ Есть в наличии

Это значительно увеличивает вероятность дополнительных покупок по сравнению с предложением дорогих товаров.
