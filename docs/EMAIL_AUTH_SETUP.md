# Email аутентификация через Resend

Теперь клиенты могут регистрироваться и входить через email с OTP кодами. Это дешевле и удобнее чем SMS.

## Как это работает

1. **Регистрация**: Клиент вводит email → получает 6-значный код на почту
2. **Вход**: Вводит код → попадает в личный кабинет
3. **Добавление телефона**: Пост-фактум может добавить номер для заказов
4. **Fallback**: Если email не работает, код уходит в Telegram

## Настройка Resend

### 1. Регистрация на Resend

1. Перейдите на [resend.com](https://resend.com/)
2. Создайте аккаунт
3. Подтвердите email
4. Получите API ключ

### 2. Настройка домена

**Вариант А: Использовать готовый домен Resend**
- Бесплатно для тестирования
- Отправка с домена `onresend.com`

**Вариант Б: Подключить свой домен**
- Добавьте DNS записи:
  ```
  TXT: v=spf1 include:sendgrid.net ~all
  TXT: v=DKIM1; k=rsa; p=...
  CNAME: dkim._domainkey.yourdomain.com -> dkim.sendgrid.net
  ```

### 3. Переменные окружения

Добавьте в `.env.local`:

```bash
# Resend Email Service
RESEND_API_KEY=re_your_api_key_here
EMAIL_FROM=FamilyHub Market <noreply@yourdomain.com>

# JWT для сессий
JWT_SECRET=your_secret_key_here

# Опционально: Telegram fallback
TELEGRAM_BOT_TOKEN=your_bot_token
TELEGRAM_CHAT_ID=your_chat_id
```

## Процесс входа клиента

### Шаг 1: Email
- Клиент открывает `/profile`
- Вводит email адрес
- Нажимает "Получить код"

### Шаг 2: OTP
- Получает 6-значный код на почту
- Красивый HTML шаблон письма
- Код действителен 5 минут

### Шаг 3: Профиль
- После ввода кода попадает в личный кабинет
- Может добавить имя
- Видит блок для добавления телефона

### Шаг 4: Телефон (опционально)
- Нажимает "Добавить номер"
- Вводит телефон в формате +38 (0XX) XXX-XX-XX
- Номер сохраняется в профиле

## Преимущества

### Для клиента
- **Бесплатно**: Не нужно платить за SMS
- **Мгновенно**: Код приходит за секунды
- **Удобно**: Email всегда под рукой
- **Безопасно**: Код действителен 5 минут

### Для бизнеса
- **Дешево**: $0.001 за email vs $0.05 за SMS
- **Надежно**: 99% доставка писем
- **Масштаб**: Неограниченное количество пользователей
- **Аналитика**: Статистика доставки в Resend

## Fallback система

Если Resend недоступен:
1. Автоматически отправляется в Telegram
2. Администратор может переслать код клиенту
3. 100% гарантия доставки

## Безопасность

- **JWT токены**: Надежные сессии на 30 дней
- **HTTP-only cookies**: Защита от XSS
- **Rate limiting**: Ограничение запросов
- **OTP TTL**: Коды действительны 5 минут
- **Валидация**: Проверка формата email

## Тестирование

3. Откройте `/profile`
4. Введите тестовый email
5. Проверьте получение письма

## Стоимость

- **Resend**: $0.001 за email (1000 писем = $1)
- **SMS**: $0.05 за SMS (1000 SMS = $50)
- **Экономия**: 98% дешевле чем SMS
