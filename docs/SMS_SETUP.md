# Настройка SMS аутентификации для личного кабинета

Теперь клиенты могут автоматически получать OTP коды по SMS, без необходимости настройки Telegram для каждого пользователя.

## Как это работает

1. Клиент вводит номер телефона в личном кабинете
2. Система генерирует 6-значный код и отправляет его по SMS
3. Клиент вводит код и получает доступ к профилю
4. Если SMS не доставился, есть fallback на Telegram для администратора

## Поддерживаемые SMS провайдеры

### 1. TurboSMS (рекомендуется для Украины)

**Плюсы:**
- Украинский провайдер, хорошая доставка по Украине
- Доступные цены
- Простая настройка

**Настройка:**
1. Зарегистрируйтесь на [turbosms.ua](https://turbosms.ua/)
2. Пополните баланс
3. Получите API токен в настройках
4. Создайте имя отправителя (сигнатуру)
5. Добавьте переменные в `.env.local`:

```bash
SMS_PROVIDER=turbosms
TURBOSMS_TOKEN=ваш_api_токен
TURBOSMS_SENDER=ваше_имя_отправителя
```

### 2. Twilio (международный)

**Плюсы:**
- Международный провайдер
- Надежная доставка
- Поддержка многих стран

**Настройка:**
1. Зарегистрируйтесь на [twilio.com](https://www.twilio.com/)
2. Получите Account SID, Auth Token и номер телефона
3. Добавьте переменные в `.env.local`:

```bash
SMS_PROVIDER=twilio
TWILIO_ACCOUNT_SID=ваш_account_sid
TWILIO_AUTH_TOKEN=ваш_auth_token
TWILIO_PHONE_NUMBER=+1234567890
```

## Переменные окружения

Обязательно добавьте в `.env.local`:

```bash
# Выбор провайдера (turbosms или twilio)
SMS_PROVIDER=turbosms

# Для TurboSMS
TURBOSMS_TOKEN=your_api_token_here
TURBOSMS_SENDER=your_sender_name_here

# Для Twilio (альтернатива)
TWILIO_ACCOUNT_SID=your_account_sid_here
TWILIO_AUTH_TOKEN=your_auth_token_here
TWILIO_PHONE_NUMBER=+1234567890

# JWT секрет для сессий
JWT_SECRET=ваш_секретный_ключ
```

## Тестирование

После настройки:

1. Перезапустите приложение
2. Откройте `/profile`
3. Введите номер телефона
4. Проверьте получение SMS с кодом
5. Введите код для входа

## Fallback система

Если SMS сервис недоступен:
- Система автоматически попытается отправить код в Telegram
- Администратор сможет переслать код клиенту
- Это обеспечивает 100% доставку кода

## Стоимость

- **TurboSMS**: ~1-2 грн за SMS в Украину
- **Twilio**: зависит от страны, ~$0.05-0.10 за SMS

## Безопасность

- Коды действуют 5 минут
- Ограничение на количество запросов с одного IP
- Ограничение на количество запросов для одного номера
- HTTP-only cookies для сессий

## Рекомендации

1. **Используйте TurboSMS** для украинских клиентов
2. **Настройте fallback на Telegram** для надежности
3. **Мониторьте баланс** SMS провайдера
4. **Проверьте имя отправителя** перед запуском

Теперь любой клиент может самостоятельно зарегистрироваться и войти в личный кабинет без помощи администратора!
